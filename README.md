# PROPROFISSÃO – Repositório de Estudos

<img src="logo-proa.svg" alt="logo do proa">

Este repositório reúne exercícios, projetos e anotações desenvolvidos durante o curso **PROPROFISSÃO**, uma formação em programação oferecida pelo Instituto PROA.

O objetivo deste repositório é registrar minha evolução técnica, consolidar conhecimentos adquiridos ao longo do curso e documentar projetos desenvolvidos durante a formação.


---

## Sobre o PROPROFISSÃO

O **PROPROFISSÃO** é um curso profissionalizante gratuito voltado para jovens que desejam ingressar na área de tecnologia e desenvolvimento de software. A formação tem duração aproximada de **6 meses** e combina aprendizado técnico com desenvolvimento profissional. ([qa-www.proa.org.br][1])

O programa prepara os alunos para atuar no mercado de tecnologia por meio de:

* Ensino de **programação e desenvolvimento de sistemas**
* Desenvolvimento de **projetos práticos**
* Preparação para o **mercado de trabalho**
* Desenvolvimento **comportamental e comunicacional**

O curso é organizado pelo **Instituto PROA**, uma organização que tem como objetivo promover inclusão produtiva e ampliar oportunidades de empregabilidade para jovens por meio da educação e da tecnologia. ([conteudo.proa.org.br][2])

---

## Estrutura do Curso

Durante a formação, os participantes têm contato com diferentes áreas relacionadas ao desenvolvimento de software e carreira em tecnologia.

### Programação e Desenvolvimento

Introdução à lógica de programação e desenvolvimento de sistemas, podendo envolver tecnologias como:

* Java
* Python
* C# / .NET
* Kotlin

Essas tecnologias são utilizadas em atividades práticas e projetos ao longo do curso. ([PROA][1])

### Desenvolvimento de Projetos

Aplicação dos conhecimentos técnicos em projetos práticos, incluindo:

* desenvolvimento de aplicações
* resolução de problemas reais
* trabalho em equipe
* organização de código e versionamento

### Preparação para o Mercado

Além da parte técnica, o curso também oferece:

* desenvolvimento de habilidades de comunicação
* preparação para entrevistas
* construção de perfil profissional (ex.: LinkedIn)

### Formação Complementar

O programa também incentiva a ampliação do repertório cultural e profissional por meio de atividades extracurriculares e eventos relacionados à tecnologia. ([PROA][1])
