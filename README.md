# PROPROFISSÃO – Repositório de Estudos

<img src="logo-proa.svg" alt="logo do proa">

Este repositório reúne exercícios, projetos e anotações desenvolvidos durante minha formação em programação no **PROPROFISSÃO**, curso oferecido pelo Instituto PROA.

O objetivo deste repositório é documentar minha evolução como desenvolvedor, registrar os conhecimentos adquiridos ao longo do curso e organizar projetos desenvolvidos durante a formação.

---

## Sobre o PROPROFISSÃO

O **PROPROFISSÃO** é um curso profissionalizante gratuito de tecnologia voltado para jovens que desejam iniciar carreira na área de desenvolvimento de software.

A formação tem duração aproximada de **6 meses** e combina aprendizado técnico, desenvolvimento de projetos e preparação para o mercado de trabalho. ([qa-www.proa.org.br][1])

O programa é organizado pelo Instituto PROA em parceria com o Senac e tem como objetivo capacitar jovens para atuar na área de tecnologia, oferecendo formação técnica e desenvolvimento de habilidades profissionais. ([Hora Brasil][2])

Durante o curso, os participantes têm contato com programação, desenvolvimento de projetos e atividades voltadas à empregabilidade e comunicação profissional. ([PROA][1])

---

## Formação – Turma C#

Estou participando da **trilha de desenvolvimento utilizando C# e .NET**, com foco em desenvolvimento de aplicações web e fundamentos de programação.

Durante o curso serão abordadas tecnologias e conceitos essenciais para o desenvolvimento full stack.

## Tecnologias e Ferramentas

Durante a formação, estou estudando as seguintes tecnologias:

### Backend

* **C#**
* **ASP.NET Core**

Essas tecnologias são utilizadas para desenvolvimento de aplicações backend e APIs.

### Frontend

* **HTML**
* **CSS**
* **JavaScript**
* **React**

Essas ferramentas são utilizadas para desenvolvimento de interfaces web modernas.

### Banco de Dados

* **SQL**
* **SQL Server**

Utilizados para modelagem, manipulação e gerenciamento de dados em aplicações.

### Design e Prototipação

* **Figma**

Utilizado para criação de protótipos de interfaces e planejamento visual de aplicações.
